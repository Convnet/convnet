## ConvnetClient ##

**Convnet** 是基于TUN-TAP驱动的使用IM形式组织用户关系的P2P架构的VPN系统。

### 1、VPN+ ###

Convnet实现了传统的基于服务器VPN的通讯功能，在此基础上实现了基于UPNP以及NAT打洞的P2P的通讯方式。
